Info

Users:
They can log into the system and perform actions like comment and rate movies. The option is available by clicking the title of the movie that they want to rate.
Also they can search for special terms displayed in the descriptions of the movies using de search bar in the top of the page.

Administrators:
They can do the same as users but also add new movies and edit the existing ones.
By filling the title of a movie and clicking the special button cloud_download they are available to use a web service from OMDbAPI in order to fill all the fields for the movie specified by title.
If a User want to become Administrator it has to go to the special page and send the special code "2YMZkWs3NDQSBVwyDKdv".
